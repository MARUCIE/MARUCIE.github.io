# Human phoneme audio attribution

These files are local offline copies of human recordings from Wikimedia Commons. The app must not describe a browser-generated voice as one of these recordings.

The machine-readable file-by-file list is in `phonemes/manifest.json`. Each entry includes the Commons source page, direct download URL, author credit, recorded IPA target, license, license link, byte size, and SHA-256 checksum. The local playback copies are transcoded from the original Ogg Vorbis recordings to MP3 for iPad/Safari compatibility; this format conversion does not replace or weaken the source license.

## Main source groups

- IPA vowel reference recordings: Denelson83 and Wikimedia Commons contributors, generally offered under CC BY-SA 3.0. See each linked Commons file page in the manifest for the controlling terms and revision history.
- IPA consonant reference recordings: Peter Isotalo and Wikimedia Commons contributors, generally offered under CC BY-SA 3.0 (with historical GFDL dual-licensing on many files). This project selects CC BY-SA 3.0 for reuse.
- General American `/eɪ/`, `/aɪ/`, and `/oʊ/`: Muke Tever, released to the public domain on the linked Commons file pages.
- U.S. English `/aʊ/` (“ow”): Dvortygirl, CC BY-SA 3.0.
